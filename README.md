# Pubvana Site Health

**I noticed folks downloading some of these packages. I'm super grateful, Thank You!  I would like to let folks know until this notice disappears I'm doing a lot of breaking changes without worrying about them.  Once versions are up around 0.5.x things should settle down.**

Site health and diagnostics package for Pubvana. Surfaces configuration and security problems early so admins can fix issues before they become incidents.

## Requirements

- PHP 8.1+
- `enlivenapp/flight-school` ^0.3
- `enlivenapp/flight-settings` ^0.2
